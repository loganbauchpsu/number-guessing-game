# Number Guessing Game

This is a python game where the player guesses a randomly generated number.

## Features
- 3 difficulty levels (easy, medium, hard)
- There are a certain amount of attempts based on difficulty
- Makes sure that guesses and numbers and are in the given range (based on difficulty)

## How to Run
1. Make sure Python is installed
2. Run:
   python num_guess.py
